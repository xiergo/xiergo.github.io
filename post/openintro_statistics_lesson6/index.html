Linear Relationship

$$ y=\beta x+\alpha $$ $\alpha$ intercept

$\beta$ slope

$x$ explanatory / predictor variable

$y$ response variable

r 函数 abline中，a = $\alpha$ , b=$\beta$

downward linear trend: 负相关

Non-linear Model

抛物线 $y=-x^2$

if we use a linear model to fit it , we have NO TREND

the "hat" in $\hat{poverty}=64.78-0.62\times HS_{grad}$ is used to signify that this is an estimate.

Residuals

residuals are the leftover variation in the data after accounting for the model fit: $$ Data=Fit+Residual $$

residual : $e_i=y_i-\hat{y_i}$

residual plot: residual ~ x

Correlation

$$ X=\left( \begin{matrix}x_1\newline x_2 \newline \vdots\newline x_n\end{matrix}\right) \ \ \ Y=\left(\begin{matrix}y_1\newline y_2\newline \vdots \newline y_n\end{matrix}\right) \\
X_1=X-\overline{X}; \ \ \ Y_1=Y-\overline{Y} $$

$$ \begin{align} correlation(X,Y)&=cos(X_1,Y_1)\newline &=\frac{X_1Y_1}{|X_1||Y_1|}\newline &=\frac{\sum_{i=1}^n(x_i-\overline{x})(y_i-\overline{y})} {\sqrt{\sum_{i=1}^n(x_i-\overline{x})^2\sum_{i=1}^n(y_i-\overline{y})^2}} \end{align} $$

if $y_i=ax_i+b$, then $R=sign(a)$

$-1\le R \le 1$

Rank correlation (spearman)

spearman correlation 就是correlation(rank(x),rank(y))

spearman: non-linear relationship
